Configuring license servers details on thin client based user VMs

Generally the client software for any license manager works with IP address and port number details of the license server.

But for some license managers like Flex LM, the host name to IP address mapping will be validated by a DNS or etc/hosts file lookup on the server/client.

Because the DNS for VDI user VMs(AD/ADC) is different than the campus DNS servers(10.1.1.8/10.1.11.1), it may be required to

1.        enter the host name/IP address of the license server in the User VM’s etc/hosts file and

2.        the User VM’s name/IP address in the license server’s etc/hosts file.
